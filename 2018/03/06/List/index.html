List集合

    List集合有ArrayList，Vector，LinkedList。这三个类都在java.util包中。

ArrayList和Vector

    他们会在内存中开辟一块连续的空间来存储，存储的数据是连续的，所以它们支持使用序号即下标来访问元素，同时索引数据的速度比较快。但是由于插入数据需要移动其他数据，所以插入数据比较慢。ArrayList和Vector都有一个初始化的空间容量大小，当超过这个大小的时候，就会动态的扩充他们的存储空间。

    相同点：

  • 内存中连续存储
  • 插入数据速度慢
  • 动态扩充存储空间

    ArrayList最大的区别时对synchronization的使用。ArrayList是不可以同步的，而Vector大多数方法时可以同步的，所以，ArrayList时线程不安全的，Vector是线程安全的。

    Vector可以同步的方法有：

  • add
  • insert
  • remove
  • set
  • equals
  • hashcode

    不同点：

  • Vector线程安全，ArrayList线程不安全

LinkedList

    LinkedList是采用双向列表来实现的，对数据的索引是从数据头开始遍历，因此用于随机访问则效率比较低，但是插入数据效率比较高。此外，LinkedList是非线程安全的。

    LinkedList特点:

  • 线程不安全
  • 插入数据效率高

如何选择使用

    当我们只是进行简单的数据存储或者再列表末尾添加删除数据，那么可以使用ArrayLsit和Vector。如果需要插入数据，则可以使用linkedList。如果要求线程安全，那么可以使用Vector。
