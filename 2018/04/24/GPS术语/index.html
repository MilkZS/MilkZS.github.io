GPS术语

Posted on 2018-04-24 | In GPS

GPS术语

AFLT

advanced forward-link trilateration 先进的前向链路三边测量

ALE

Aggregate Location Estimator,综合位置估算器

Almanac

历书也包含了卫星的位置等相关信息,不过它是星历数据的简化集,其
精度较低。所以,历书数据量较小,传输时间较短。

C/A码

粗捕获码。普通用户用来测定测站到卫星距离的一种主要信号

C/No

卫星的信噪比，必须大于28才可用

DPO

Dynamic Power Optimization ,动态功耗优化

Ephemeris

星历表本来是用来记录天体特定时刻的位置的。而在 GNSS 中,星
历表则记录了卫星的一些运行参数,它使得我们通过星历表就可以计算出任意时刻的导航卫星的位置和
速度。我们将见到在 GPS 中,星历表包含了非常详细的卫星轨道和位置信息,所以其数据量较大,传输
时间较长。为了克服这个问题,人们设计了星历表的简化集,即历书。

GNSS

Global Navigation Satellite System,全球导航卫星系统

GPS

Global Positioning System, 全球定位系统

LBS

Location Based Service, 基于用户的位置信息为用户提供服务的增值业务

TTFF

首次定位时间

UTC

协调时间时，又称世界统一时间、世界标准时间、国际协调时间。

SV

satellite value，卫星信息的简称。

TLE

卫星星历

SAP

Sensor-assisted positioning 传感器辅助定位

SBAS

Satellite Based Augmentation Systems,利用地球静止轨道卫星建立的地区性广域差分增强系统。

MO

Mobile-Originated

MT

Mobile-Terminated

NLP

Network Location Provider

GTP

Global Terrestrial Positioning,全球地面定位

UTC

世界统一时间
